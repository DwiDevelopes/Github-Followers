# Github-Followers
Github Followers Real Indonesia

# Installer
`pip install PyGithub`

# Pelanggran

Untuk membuat skrip Python yang dapat menambahkan followers ke akun GitHub secara otomatis, penting untuk diketahui bahwa hal tersebut melanggar Terms of Service GitHub. GitHub memiliki kebijakan yang ketat terhadap manipulasi followers, likes, atau aktivitas lainnya yang tidak organik. Melakukan hal tersebut dapat mengakibatkan akun Anda di-banned atau di-suspend.

- GitHub Token: Anda perlu membuat Personal Access Token (PAT) dari GitHub. Buka Settings > Developer settings > Personal access tokens dan buat token dengan izin yang sesuai.

- Etika dan Legalitas: Pastikan Anda menggunakan API GitHub untuk tujuan yang sah dan sesuai dengan kebijakan GitHub.

- Manipulasi Followers: Menambahkan followers secara otomatis atau tidak organik adalah pelanggaran dan dapat mengakibatkan konsekuensi serius.
